# Nexo AI

A powerful AI assistant built by the **Nexo Mind Team**.

## Features

- AI chat powered by Llama 4 Scout via Groq
- Multi-turn conversation history
- Beautiful dark UI with animated landing page
- Sinhala language support
- Authentication via Hercules Auth

## Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS 4, Vite
- **Backend**: Convex (database + serverless functions)
- **AI**: Groq / Llama 4 Scout via Hercules AI Gateway
- **Auth**: Hercules Auth (OIDC)
